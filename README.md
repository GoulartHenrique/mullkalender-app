# 🗑️ Müllkalender App

Web app para consulta de calendário de coleta de lixo em Erfurt com assistente IA.

## Stack
- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Node.js + Express + TypeScript
- **Database:** MongoDB Atlas
- **AI:** Claude API

## Setup
```bash
# Backend
cd backend && npm install

# Frontend  
cd frontend && npm install
```
