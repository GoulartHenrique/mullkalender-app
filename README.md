# 🗑️ Müllkalender App

Web app for checking the trash collection schedule in Erfurt with an AI assistant.

## Stack

- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Node.js + Express + TypeScript
- **Database:** MongoDB Atlas
- **AI:** Claude API

## Setup

```bash
# Backend
cd backend && npm install

# Frontend
cd frontend && npm install
```
